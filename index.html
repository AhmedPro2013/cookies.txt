Chrome export cookie.txt

Google Chrome extension

View/Fork On GitHub
  • Install from Google Chrome Web Store

    Allows users to export their session cookie data from Chrome to a legacy Netscape/Mozilla file format so wget (and other tools) can use it.

    This allows wget to continue a possibly authenticated session that was started in a Chrome.

    Installation

    Google Chrome Web Store
